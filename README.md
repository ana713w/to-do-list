# Mis Tareas

App de lista de tareas construida con React, TypeScript y Tailwind CSS.

## Tecnologías

- React 19 + TypeScript
- Tailwind CSS v4
- Vite
- Vitest + Testing Library

## Instalación

```bash
npm install
npm run dev
```

## Scripts

| Comando | Descripción |
|---|---|
| `npm run dev` | Servidor de desarrollo |
| `npm run build` | Build de producción |
| `npm test` | Ejecutar tests |
| `npm run preview` | Vista previa del build |

## Funcionalidades

- Agregar tareas con título y descripción opcional
- Marcar tareas como completadas
- Eliminar tareas
- Filtrar por todas / pendientes / completadas
